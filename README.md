### Install

1. `bundle install`

### advent_of_code_cli

1. `export AOC_COOKIE={{cookie_from_browser_after_logging_in}}`
2. `bundle exec aoc_cli scaffold {{day_number}}`
3. `bundle exec aoc_cli download {{day_number}}`
4. `bundle exec aoc_cli solve {{day_number}}`

### Running tests

1. `ruby 01.rb`
